# 🚦 Traffic Light Web App

A simple interactive traffic light built using **HTML, CSS, and JavaScript**.

## 📌 Features

- Click **STOP** → Red light turns on
- Click **READY** → Yellow light turns on
- Click **GO** → Green light turns on
- Only one light is active at a time
- Clean and beginner-friendly code

## 🛠️ Technologies Used

- HTML
- CSS (Flexbox for layout)
- JavaScript (DOM Manipulation)

## 🎯 What I Learned

- How to structure a web page using HTML  
- Styling using CSS Flexbox  
- Using JavaScript functions  
- Handling click events  
- Changing styles dynamically with JavaScript
